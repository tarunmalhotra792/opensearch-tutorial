Welcome to Learn OpenSearch

Welcome to the world of OpenSearch, a robust and flexible search engine toolkit designed to help you search, visualize, and analyze your data in real-time.

Originally developed as a fork of Elasticsearch and Kibana, OpenSearch was created by Amazon Web Services (AWS) and is governed by the open-source community. OpenSearch consists of a search engine (OpenSearch) and a visualization tool (OpenSearch Dashboards), which work together to help organizations build scalable, high-performance search and analytics solutions.
